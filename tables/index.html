Tags in HTML

In HTML, a table is used to display data in a structured format of rows and columns, similar to a spreadsheet. It is created using the <table> element, which organizes information into rows (<tr>) and cells (<td> for normal data and <th> for headings). Tables are especially useful when you need to present related data side by side, such as schedules, price lists, or comparison charts. A table usually starts with an optional header row that defines the column titles, followed by multiple rows of data. You can also group sections of a table with <thead>, <tbody>, and <tfoot> for better structure and styling.

Table Tags used in HTML

1. <table>

Defines a table container. 

<table>
  <!-- table rows go here -->
</table>

2. <tr>

Defines a table row. 

<tr>
  <!-- table cells go here -->
</tr>

3. <td>

Defines a table data cell (regular cell). 

<td>Cell content</td>

4. <th>

Defines a table header cell (bold, centered by default). 

<th>Header</th>

5. <caption>

Adds a caption (title) to the table. 

<caption>Table Title</caption>

6. <thead>

Groups header rows in a table. 

<thead>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
</thead>

7. <tbody>

Groups body content rows in a table. 

<tbody>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</tbody>

8. <tfoot>

Groups footer rows in a table. 

<tfoot>
  <tr>
    <td>Footer 1</td>
    <td>Footer 2</td>
  </tr>
</tfoot>

9. <col>

Defines column properties (like width, style). 

<col>

10. <colgroup>

Groups columns for styling or attributes. 

<colgroup>
  <col span="2" style="background-color:#f2f2f2">
</colgroup>



Table Attributes used in HTML

1. <align>

Specifies horizontal alignment of the table.
  • left
  • center
  • right

<table align="center">

2. <border>

Specifies the width of the table border.
  • 0 (no border)
  • 1 (thin border)
  • 2, 3, ... (thicker borders)

<table border="1">

3. <cellpadding>

Specifies the space between the cell content and cell border.
  • 0
  • 5
  • 10
  • any positive number

<table cellpadding="5">

4. <cellspacing>

Specifies the space between table cells.
  • 0
  • 5
  • 10
  • any positive number

<table cellspacing="5">

5. <width>

Specifies the width of the table.
  • Percentage (e.g., 50%)
  • Pixels (e.g., 300px)

<table width="50%">

6. <height>

Specifies the height of the table.
  • Percentage (e.g., 50%)
  • Pixels (e.g., 200px)

<table height="200px">

7. <background>

Specifies a background image for the table.
  • URL of an image

<table background="bg.jpg">

8. <bordercolor>

Specifies the color of the table border.
  • color names (e.g., red)
  • hex values (e.g., #ff0000)

<table border="1" bordercolor="red">

9. <bgcolor>

Specifies background color of the table.
  • color names (e.g., yellow)
  • hex values (e.g., #ffff00)

<table bgcolor="yellow">

10. <colspan>

Merges multiple columns into a single cell.
  • Any positive integer (e.g., 2, 3, 4)

<td colspan="2">Merged Cell</td>

11. <rowspan>

Merges multiple rows into a single cell.
  • Any positive integer (e.g., 2, 3, 4)

<td rowspan="2">Merged Cell</td>

12. <align>

Specifies horizontal alignment of the cell content.
  • left
  • center
  • right

<td align="center">Centered Text</td>

13. <valign>

Specifies vertical alignment of the cell content.
  • top
  • middle
  • bottom

<td valign="middle">Middle Aligned</td>
